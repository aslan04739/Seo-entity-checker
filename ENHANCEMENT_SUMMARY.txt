## 📊 What Gets Saved

### Configuration File
Location: `~/.seo_crawler/config.json`

```json
{
  "max_pages": 10,
  "max_depth": 2,
  "creds_path": "/path/to/your/credentials.json",
  "recent_urls": [
    "https://www.example.com",
    "https://www.bbc.com",
    "https://www.wikipedia.org",
    ...
  ]
}
```

### Analysis Output
Location: Your chosen directory (usually Downloads)

```csv
source,name,salience,category
https://example.com,London,0.85,LOCATION
https://example.com/news,BBC,0.92,ORGANIZATION
https://example.com/sports,World Cup,0.78,EVENT
```

## 🔧 For Power Users

### Command-Line Mode (Headless)
```bash
python3 analyze2.py \
  --url https://example.com \
  --creds ~/.seo_crawler/key.json \
  --pages 10 \
  --depth 2 \
  --out results.csv
```

### View Your Configuration
```bash
cat ~/.seo_crawler/config.json
```

### Reset Everything
```bash
rm -rf ~/.seo_crawler/
# Restart app - fresh settings!
```
